Modern Architecture

VerMind implements state-of-the-art transformer architecture with optimizations for both training efficiency and inference speed.

  • Grouped Query Attention (GQA) for memory efficiency
  • RMSNorm for stable training
  • Flash Attention support for faster computation
  • Distributed Data Parallel (DDP) training
  • Mixed precision training (bfloat16/float16)
  • Automatic checkpoint management

Quick Start

Get up and running in minutes

1

Clone & Install

git clone https://github.com/nev8rz/vermind.git cd vermind uv venv && source .venv/bin/activate uv pip install -e .
2

Pre-training

python train/pretrain.py \ --data_path /path/to/data.jsonl \ --save_dir ./output/pretrain \ --hidden_size 768 \ --num_hidden_layers 16
3

Fine-Tuning (SFT or LoRA)

python train/sft.py \ --data_path /path/to/sft_data.jsonl \ --from_weight ./output/pretrain/checkpoint
4

Deploy with vLLM

python vllm_adapter/start_server.py /path/to/checkpoint # Server starts at http://localhost:8000
