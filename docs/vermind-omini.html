Quick Start

Deploy VerMind-OMini on your edge device

1

Install

pip install vermind-omini
2

Load Quantized Model

from vermind_models import VerMindOMini model = VerMindOMini.from_pretrained( "nev8rz/vermind-omini", load_in_8bit=True # or load_in_4bit=True )
3

Export to ONNX (Optional)

python scripts/export_onnx.py \ --model_path ./output/omini \ --output_path ./output/omini.onnx \ --quantize int8
4

Deploy on Mobile

# iOS (Core ML) python scripts/export_coreml.py --model_path ./output/omini # Android (TFLite) python scripts/export_tflite.py --model_path ./output/omini
