Quick Start

Get started with VerMind-V in minutes

1

Installation

git clone https://github.com/nev8rz/vermind.git cd vermind uv venv && source .venv/bin/activate uv pip install -e .
2

Visual Pre-training

bash examples/vlm_pretrain.sh # or python train/vlm_pretrain.py \ --data_path /path/to/image_text_pairs.jsonl \ --vision_encoder_path /path/to/siglip
3

Visual Instruction Tuning

bash examples/vlm_sft.sh # or python train/vlm_sft.py \ --data_path /path/to/visual_instructions.jsonl \ --from_weight ./output/vlm_pretrain
4

Deploy

python vllm_adapter/start_server.py ./output/vlm/vlm_base # Server: http://localhost:8000
